SWIM DOJO WORKOUT GENERATOR
===========================
Simple Instructions for Beginners

WHAT IS THIS?
This program gets you a random swim workout from SwimDojo.com 
with just a few clicks. No typing or coding needed!

FIRST TIME SETUP (Takes 5 minutes)
----------------------------------

Step 1: Install Python
  - Go to: python.org/downloads
  - Click the big yellow download button
  - Run the installer
  - IMPORTANT: Check the box "Add Python to PATH"
  - Click "Install Now"
  - Wait... Done!

Step 2: Download This App
  - You probably already did this if you're reading this file
  - Make sure all files are in one folder

Step 3: Run the App
  - Find the file named: run_workout_generator.bat
  - Double-click it
  - First time takes a minute to install stuff
  - A window will pop up!

USING THE APP
-------------

1. The app window opens automatically
2. You'll see a dropdown menu with workout types
   - Pick one, or leave it on "Random"
3. Click the big "Get Random Workout!" button
4. A workout page opens in your web browser
5. That's it!

WANT A DESKTOP ICON?
--------------------
Double-click: create_desktop_shortcut.bat
An icon will appear on your Desktop
