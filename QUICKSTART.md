Follow these steps to set up and use the Swim Dojo Workout Generator.

## Step 1: Install Python (One-Time Setup)

1. Visit https://www.python.org/downloads/
2. Download the latest version of Python for Windows.
3. Run the installer.
4. Ensure the "Add Python to PATH" option is checked before proceeding.
5. Click "Install Now" and wait for the installation to finish.
6. Close the installer when complete.

## Step 2: Download the Application

1. Go to https://github.com/michal-nak/swim_dojo_generator
2. Click the "Code" button and select "Download ZIP".
3. Save the ZIP file and extract it to a preferred location (e.g., Desktop or Documents).

## Step 3: Run the Application

1. Open the extracted folder.
2. Double-click `run_workout_generator.bat`.
3. If prompted for permission, select "Yes".
4. On first run, required packages will be installed automatically (this may take about a minute).
5. The Swim Workout Generator window will open.

## Step 4: Generate a Workout

1. Optionally, select a workout type from the dropdown menu, or leave it as "Random".
2. Click the "Get Random Workout!" button.
3. A workout page will open in your web browser.

## Optional: Create a Desktop Shortcut

To create a desktop shortcut for easier access:

1. In the extracted folder, double-click `create_desktop_shortcut.bat`.
2. If prompted, click "Yes" to allow changes.
3. A shortcut named "Swim Workout Generator" will appear on your Desktop.
4. You can now launch the application directly from the desktop icon.

## Troubleshooting

### "Python is not recognized..."
- Ensure Python is installed (see Step 1).
- Confirm that "Add Python to PATH" was selected during installation.
- If Python was installed without this option, uninstall and reinstall with the option enabled.

### "The system cannot find the file specified"
- Verify you are in the correct extracted folder.
- Ensure you are running `run_workout_generator.bat`.

### Application window does not appear
- Check if the window is behind other open windows.
- Try closing all windows and running the application again.
- If issues persist, run `setup.bat` first, then `run_workout_generator.bat`.

### No workout opens in browser
- Check your internet connection.
- Try clicking the button again.
- Review the status log at the bottom of the application window for error messages.

## Tips

- You can run the application multiple times to generate different workouts.
- Keeping the application window open allows for quick access to new workouts.
- The "Refresh Tags" button updates the list of available workout types.
