# Quick Start Guide - Swim Dojo Workout Generator

This guide will help you get started in just a few minutes!

## Step 1: Install Python (One-Time Setup)

1. Go to https://www.python.org/downloads/
2. Click the big yellow "Download Python" button
3. Run the installer
4. **IMPORTANT**: Check the box that says "Add Python to PATH" at the bottom of the installer
5. Click "Install Now"
6. Wait for installation to complete
7. Click "Close"

## Step 2: Download This Application

1. Go to https://github.com/michal-nak/swim_dojo_generator
2. Click the green "Code" button
3. Click "Download ZIP"
4. Save the ZIP file to your Downloads folder
5. Right-click the ZIP file and select "Extract All"
6. Choose a location (like your Desktop or Documents folder)
7. Click "Extract"

## Step 3: Run the Application

1. Open the extracted folder
2. Double-click the file named `run_workout_generator.bat`
3. A window might pop up asking "Do you want to allow this app to make changes to your device?" - Click "Yes"
4. The first time you run it, it will install some required packages (this takes about 1 minute)
5. The Swim Workout Generator window will open!

## Step 4: Get Your Workout

1. (Optional) Choose a workout type from the dropdown menu, or leave it as "Random"
2. Click the big "Get Random Workout!" button
3. A workout page will open in your web browser
4. Enjoy your swim training!

## Troubleshooting

### "Python is not recognized..."
- You need to install Python (see Step 1)
- Make sure you checked "Add Python to PATH" during installation
- If you already installed Python without this option, uninstall it and reinstall with the option checked

### "The system cannot find the file specified"
- Make sure you're in the correct folder (the one you extracted)
- Make sure you double-clicked `run_workout_generator.bat` (not any other file)

### Application window doesn't appear
- Check if a window appeared behind your other windows
- Try closing all windows and running again
- Try running `setup.bat` first, then `run_workout_generator.bat`

### No workout opens in browser
- Check your internet connection
- Try clicking the button again
- The status log at the bottom of the window will show error messages if something goes wrong

## Tips

- You can run the application as many times as you want to get different workouts
- Leave the application window open to get multiple workouts quickly
- The "Refresh Tags" button updates the list of available workout types
